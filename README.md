## 拓展功能
### 系统功能
+ 解除录屏/截图限制
+ 过VPN检测
### 游戏功能
+ 王者荣耀 过新号实名认证 
+ 网易游戏 强行登录ios(部分)
### 应用功能
+ 快手/快手极速版 无水印保存视频
+ TIM 拓展功能
+ 微信 拓展功能
+ 奇妙应用 自动签到
+ App分享 去开屏广告 自动签到
### 其他功能
+ 去除部分弹窗(云注入弹窗/清羽弹窗)
+ 去除部分常见广告
+ Momo 全过/全红/检测游戏(长按表情切换形态)
